# 系统程序设计课程设计后端
基于原项目做代码重构  
## 技术栈
> SpringBoot2.0 \+ MySql 5.6 \+ Lombok等  
> 接口按照标准REST风格进行设计,[参考博客](https://www.jianshu.com/p/84568e364ee8),[GitHub官方说明文档](https://developer.github.com/v3/)
## 基本功能  
* 用户注册、登录模块  
* 个人信息管理模块  
* 管理员对用户信息、监考信息、任务信息的增删改查模块  
* 用户查询监考信息、任务信息的模块等等  ## E-R图
